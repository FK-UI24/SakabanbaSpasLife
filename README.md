# SakabanbaSpasLife

## Overview
一時期はまっていたサカバンバスピスを使ったゲームです。
ひたすら敵をよけていくだけです。
敵に当たると視界が暗くなっていき、真っ暗になるとゲームオーバーになります。
操作方法は上下左右キーです。<br>
**右からくる生物 → 敵**<br>
**下から来るクリオネ → 回復**<br>
**靴 → 入力反転**<br>
**ケーキ → 一時的に無敵、サイズが少し大きくなる**<br>
※Unityプロジェクトは含まれていますが、ビルド済みアプリケーションは含まれていません。
